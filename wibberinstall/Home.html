Installation

  1. Signup for Tibber.
  2.  Create an Access Token for your account.
  3.  Install Wibber.
  4. Make sure a Nickname is associated with your home in Tibber. This is created in the Tibber iPhone app: My Account -> My Home -> Nickname.
  5. Enter Tibber Access token in Wibber

The easisest way to enter the Token, is to create a Text Replacement for it on the Iphone. This is done in Settings->General->Keyboard->Text Replacement.
